Paper summary

Abstract

Universal vision-language-action models are driving embodied intelligence toward multi-platform unification. However, existing strategies predominantly learn cognitive intents rather than physical execution boundaries, leaving robots highly vulnerable to kinematic violations and task crashes during morphological shifts. Here we report a physics-internalizing foundation framework that decodes physical schemas into executable constraints for zero-shot robot control. To achieve this, we developed EmbodiAtlas-110K, a configuration engine providing 110,602 parsable, composable, and perturbable body-action environments. By learning how physical structures dictate motion boundaries within this engine, the framework dynamically routes task intentions to feasible kinematic chains. This enables unseen robots to be controlled solely via standard description files without target demonstrations. Evaluated within StarVLA, our method increased zero-shot success in a 6-to-7 degree-of-freedom transfer from 8.1% to 44.8% against structure-free baselines, and tripled performance across 36 unseen arm-gripper combinations. Furthermore, the optimal configuration achieved an 81.5% success rate on LIBERO-Plus, substantially outperforming state-of-the-art models including π0.5 and ACoT-VLA. These results indicate that internalizing physical descriptions into kinematic constraints is highly effective for zero-shot adaptation under severe structural shifts, providing a practical, target-data-free deployment solution for heterogeneous and reconfigurable robots.
